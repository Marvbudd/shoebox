# First Launch

Welcome to your first time opening Shoebox!

## What to Expect

When you launch Shoebox for the first time, you'll see a **sample archive** already loaded. This sample demonstrates the app's features without requiring you to set up your own data first.

## Exploring the Sample Archive

The sample archive includes:

- **Photos**: Various family photos showing metadata examples
- **Videos**: Sample video clips with descriptions
- **Audio**: Voice recordings and historical audio
- **Face Detection**: Pre-tagged faces showing the face detection feature
- **Rich Metadata**: People, places, dates, sources demonstrating the data structure

### Take a Tour

Try these actions to familiarize yourself:

1. **Navigate**: Use **↑/↓** arrow keys to move through items
2. **Sort**: Use the **Sort** dropdown at the bottom of the window to change sort order
3. **Filter**: Toggle filters with **Ctrl+Shift+P/A/V** (Photo/Audio/Video) or use checkboxes at the bottom
4. **Slideshow**: Press **Space** to start an auto-cycling slideshow
5. **Face Tags**: Press **Ctrl+Shift+F** to toggle face tag overlays
6. **Metadata**: Click the expand icon to view detailed information

### Understanding the Interface

#### Header (Top)
Shows information about the currently selected item:
- Accession number
- Date
- Location
- People
- Description

#### Navigation (Left)
Browse your archive:
- Chronological list
- Jump to specific items
- See preview thumbnails

#### Preview List (Center)
Grid view of items with:
- Thumbnails
- Basic metadata
- Selection indicator

#### Detail View (Right)
Full-size display:
- Large photo/video/audio player
- Expandable metadata panel
- Face tag overlays (for photos)

## Next Steps

Once you're comfortable with the sample archive:

1. **Plan Your Archive**: Decide which media files to include
2. **Organize Files**: Gather photos, videos, audio into folders
3. **Create Archive Structure**: See [Creating Your Archive](./creating-archive.md)
4. **Import Metadata**: Add people, places, dates, descriptions

## Sample Archive Location

The sample archive files are located in:

- **macOS**: `/Applications/Shoebox.app/Contents/Resources/app/resource/`
- **Windows**: `C:\Program Files\Shoebox\resources\app\resource\`
- **Linux**: `[AppImage mount]/resources/app/resource/`

::: warning Don't Modify Sample Files
The sample archive is part of the application bundle. Changes won't persist across updates. Instead, create your own archive following the [Creating Your Archive](./creating-archive.md) guide.
:::
