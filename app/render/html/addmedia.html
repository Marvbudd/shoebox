Use this form at your own risk and backup beforehand! This form gets new media set up - quick and very dirty.

There are three separate things you can do here:

  1. Create an accessions.json for a directory, or update it with added files.
  2. Update all media in a collection.
  3. Update an accession - same as pressing the "Edit Media" button below the media.

Select the operation you wish in the dropdown.

For 1 - Use the button to Select the directory. This will allow you to choose the directory and it will be entered for you. The Title you enter will be the new accessions.json title.

For 2 - Enter the collection short title and press Select to see if it exists.

For 3 - Enter the accession number and press Select to see if it exists.

For all - after pressing Select you then need to edit the lower fields with changes. Link and Accession are ignored, and Description, Date and Location replace existing values. The form doesn't delete information, and not all data can be displayed here. Person and Source data creates a new person or source.

Finally press Save to do the requested operation. If you need more advanced changes close this program and use a text editor to update accessions.json directly.
