Create a Collection

A new collection will be created with the above properties.

  • Key is the filename in the collections folder (keep this short with no spaces)
  • Text is the name that shows when selecting it (keep this less than 15 characters)
  • Title will show at the top of the page if it is exported and can be longer

Delete Collection

Deleting a collection renames it in the collections folder so it will be inactive: the .json extension is replaced with yearJulianDayHourMinute. You can manually restore it by changing the extension back to .json.
